# Anthumchris Tt

## How do I install these formulae?

`brew install anthumchris/tt/<formula>`

Or `brew tap anthumchris/tt` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "anthumchris/tt"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
